  1. HTML 是谁发明的

  1. HTML 起手应该写什么

!加tab按键

  1. 常用的表章节的标签有哪些

常用的表示章节的标签是

  • h1:标题1,h2:标题2,h3:标题3,h4:标题4,h5:标题5,h6:标题6,标题字体依次递减
  • section表示章节
  • article表示文章
  • main表示主要内容
  • aside表示次要内容
  • p表示段落
  • header表示头部
  • footer表示脚部
  • 用div划分

  1. 全局属性有哪些，分别是什么意思

  • class
  • contenteditable
  • hidden
  • id
  • style (加入body中style{display:block;})
  • tabindex(-1表示不允许,0表示最后一项)
  • title

  1. 常用的内容标签有哪些，分别是什么意思

  • ol+li (ordered list + list item)
  • ul+li (unordered list + list item)
  • dl+dt+dd(description list + term + data)
  • pre (preview,标签用来定义预格式化的文本，被包围在pre标签中的文本通常会保留空格和换行符， 而文本也会呈现出等宽字体。同样，pre标签的一个常见的应用便是用来保存计算机中的源代码文本)
  • hr (分隔线)
  • br(break)
  • a(anchor)
  • em(emphasis语气强调)
  • strong(内容强调)
  • code(将文本变成等宽字体；还有一个功能就是暗示这段文本是源程序代码,浏览器可以做出自己的 样式处理)
  • q(quote引用标签,表示一个封闭的并且是短的行内引用的文本. 这个标签是用来引用短的文本，所以请不要引入换行符)
  • blockquote(块级引用元素,对于长的文本的引用)
